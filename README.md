# CAMS_CAS_ReaderExtracter
CAMS_CAS_ReaderExtracter

In India CAMS is the organization which takes care of the book management of mutual funds. Retail users can get their statements from : https://www.camsonline.com/Investors/Statements/Consolidated-Account-Statement.
This code reads such statemenet PDFs and convertes them to xls


how to run

Intall the prerequisites and keep your pdfs in the Resources folder including your password file and execute.

python ReadAndExtract.py
